Title: College President Election 2025 Voting System

A simple web-based online voting system built with Firebase and Chart.js.
It allows students to vote for candidates (Devisri, Praneetha, Ruchitha) and displays real-time results in a bar chart.

🔧 Features:
One vote per user (Firebase Realtime Database)

Real-time vote count updates

Interactive bar chart (Chart.js)

Google login (if enabled)

Hosted using Firebase Hosting
